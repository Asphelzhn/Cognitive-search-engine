
# Upgrade pip
`pip install -U pip`
or if "pip is undefined"
`python3 -m pip install -U pip`

# Install virtualenv
`pip install virtualenv`

# Create virtualenv
`virtualenv venv`

# Activate virtualenv
`source venv/bin/activate`

# Install requirements
`pip install -r requirements.txt`
